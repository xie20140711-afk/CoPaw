# CoPaw - Hugging Face Spaces

🤖 基于 AgentScope 的个人 AI 助手

## 功能特性

- 💬 智能对话
- 🔧 多模型支持 (OpenAI, Anthropic, 本地模型)
- 🧠 记忆管理
- 🛠️ 自定义技能
- ⏰ 定时任务

## 快速开始

1. 点击右上角 **Duplicate Space** 复制此项目
2. 在 Settings 中添加你的 API Keys:
   - `OPENAI_API_KEY`
   - `ANTHROPIC_API_KEY`
3. 重启 Space 即可使用

## 本地运行

```bash
pip install -r requirements.txt
python app.py
```
